Quick Instructions

  1. Ensure AI assistant server is running
  2. Type commands in the console (e.g., 'suggest', 'explain')
  3. Use 'help' to see all available commands
  4. For file operations, set file path in settings
  5. Deep integration with CoffeeCup HTML Editor

Console Commands

  • suggest [code] - Get code improvements
  • explain [code] - Get code explanations
  • apply [code] - Apply changes to file
  • models - List available models
  • clear - Clear console output
  • help - Show this help
  • exit - Exit console
